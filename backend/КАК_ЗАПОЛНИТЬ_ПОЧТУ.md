# Как один раз настроить почту (коды на email)

Пароль **никому не присылай** — ни в чат, ни в тикет. Вписываешь его только у себя в файл или в скрипт.

---

## Вариант А: Скрипт сам всё спросит (проще всего)

В терминале:

```bash
cd backend
python3 setup_mail.py
```

Скрипт по очереди спросит: сервер, порт, email, пароль, URL сайта. Пароль вводишь в терминал — он запишется в `mail.env` и на экран не выведется. В конце перезапусти бэкенд.

---

## Вариант Б: Вписать вручную в файл

1. Открой файл **`backend/mail.env`** в редакторе.
2. Замени только эти строки на свои данные:

| Строка | Что вписать |
|--------|-------------|
| `SMTP_HOST=` | `smtp.gmail.com` (Gmail) или `smtp.mail.ru` (Mail.ru) или `smtp.yandex.ru` (Яндекс) |
| `SMTP_PORT=` | Обычно `587` (оставь как есть) |
| `SMTP_USER=` | Твой полный email |
| `SMTP_PASSWORD=` | Пароль от почты **или** пароль приложения (Gmail/Mail.ru/Яндекс — см. MAIL_SETUP.md) |
| `MAIL_FROM=` | Тот же email что и SMTP_USER |
| `FRONTEND_VERIFY_EMAIL_URL=` | Адрес сайта, например `https://твой-домен.ru` |

3. Сохрани файл. Перезапусти бэкенд.

---

## Откуда взять пароль приложения (если включена 2FA)

- **Gmail:** [Google → Безопасность → Пароли приложений](https://myaccount.google.com/apppasswords) — создать пароль для «Почта».
- **Mail.ru:** Настройки → Пароль и безопасность → Пароль для внешнего приложения.
- **Яндекс:** [id.yandex.ru](https://id.yandex.ru) → Пароли приложений.

Подробнее: **backend/MAIL_SETUP.md**.

После настройки проверь: зарегистрируй тестового юзера — на почту должен прийти код.
